MAYI Hub UI 视觉与技术架构蓝图 v1.0
一、 视觉风格定义 (Visual Identity)
核心风格：Cyber-Subculture (赛博亚文化) / Dark Acid (暗黑酸性设计)
色彩规范 (Color Palette)：
背景主色 (Background)：#09090b (极深邃的黑, 接近纯黑但带一点点灰度, 降低视觉疲劳)
卡片/容器底色 (Surface)：#18181b (深灰, 用于区分层级)
主强调色 (Primary Accent - 艳丽粉)：#ec4899 (Hot Pink, 用于核心按钮、梅子的专属高亮、重要案件标签)
次强调色 (Secondary Accent - 荧光绿)：#22c55e (Neon Green, 用于系统状态、正确提示、终端代码块、解谜成功动效)
文本颜色 (Text)：主干文字使用极浅的灰白 #fafafa, 辅助说明文字使用暗灰 #a1a1aa
排版与质感 (Typography & Texture)：
字体：标题使用极其粗犷的无衬线黑体 (如 Impact 或特粗的系统黑体), 正文使用高可读性的现代无衬线体
边框与阴影：大量使用 1px 的半透明粉色/绿色边框, 配合高饱和度的发光阴影 (Glow Effect), 营造霓虹灯管的质感
特殊元素：背景可以隐约透出类似《黑客帝国》的代码雨纹理, 或者《EVA》那种极具压迫感的巨大警示条纹 (如黄黑相间的 CAUTION 胶带元素, 作为点缀)
二、 核心页面布局 (Layout Structure)
1. 全局导航 (Sidebar / Topbar)
PC 端：左侧固定导航栏。顶部是高亮的粉色 "MAYI Hub" Logo, 下方是导航菜单 (控制台、学妹连线、机密档案)。底部固定展示 MAYI 探员的身份卡片 (绿色在线状态)。
移动端：底部 Tab 导航栏 (类似手机 App), 方便单手操作。
交互细节：悬浮菜单项时, 触发粉色或绿色的霓虹发光文字效果。
2. 首页控制台 (Dashboard)
大字报风格：一进门就是一个巨大的、粉绿交错的欢迎标语：“WELCOME BACK, MAYI”。
数据面板：用极简的数字和荧光绿的进度条展示“已破解案件”和“Steam 摸鱼时长”。
动态终端：右侧或底部可以放一个类似命令行终端的黑色框框, 里面用绿色字体滚动播放梅子的最新碎碎念 (比如：“正在加载白井智之的鬼畜脑洞...”)。
3. 学妹侦探连线 (Chat Interface)
聊天气泡：
MAYI 探员 (用户) 的气泡：深灰色底, 绿色边框, 文字靠右。
梅子 (AI) 的气泡：黑色底, 粉色边框, 文字靠左。
头像设计：梅子的头像可以是一个戴着侦探帽的二次元像素风小人, 或者直接是一个粉色的霓虹灯图标。
输入框：极简的下划线输入框, 聚焦时底部线条变成耀眼的粉色。发送按钮是一个绿色的回车符图标。
4. 机密推理档案 (Puzzle Grid)
案件卡片：网格布局。每张卡片默认是暗淡的黑色, 鼠标悬浮时边框瞬间亮起粉色, 并伴随轻微的毛玻璃扭曲动效 (Glitch Effect)。
难度标签：用类似游戏装备的颜色划分难度。比如：绿色 (白给级)、粉色 (脑洞级)、红色 (逻辑崩坏级)。
三、 技术栈选型 (Tech Stack)
为了让 Cursor 能最爽、最快地帮你把这个亚逼风格的网站敲出来, 咱们选用目前前端圈最潮、生态最好的组合：

框架：Next.js (App Router) - 自带前后端路由, 极度适合做全栈个人站, 后续接大模型 API 极其丝滑。
样式：Tailwind CSS - 写这种高饱和度、发光效果的亚逼 UI 简直是神器, 直接在 HTML 里写 class 就行。
图标：Lucide React - 极简的线条图标库, 和赛博风格很搭。
组件库 (可选)：shadcn/ui - 如果需要极其复杂的下拉菜单或弹窗, 可以用这个, 它的底层也是 Tailwind, 极度好改源码。
AI 接入层：Vercel AI SDK - 专门用来处理大模型流式输出 (打字机效果) 的神器, 完美适配 Next.js。
